\# 📈 Stock Trading Platform



\## Description

The Stock Trading Platform is a Java console-based application that simulates a basic stock trading system. Users can view available stocks, buy and sell shares, manage their portfolio, and track transaction history using Object-Oriented Programming (OOP).



\## Features

\- View available stocks

\- Buy stocks

\- Sell stocks

\- View portfolio

\- Transaction history

\- OOP-based implementation



\## Technologies Used

\- Java

\- Object-Oriented Programming (OOP)

\- ArrayList

\- HashMap

\- Scanner



\## Files

\- Main.java

\- Stock.java

\- StockMarket.java

\- Portfolio.java

\- Transaction.java



\## How to Run



Compile the project:



```bash

javac \*.java

```



Run the project:



```bash

java Main

```
